# Reference Materials

This folder contains reference materials that serve as design models for the Investor Dossier system.

## Files

### `yoko-li-reference-dossier.html`

**Purpose**: This is the original, hand-crafted Yoko Li investor dossier that serves as the gold standard for all generated dossiers.

**Why It's Important**:
- Demonstrates the ideal structure and organization of investor information
- Shows the level of detail and source citation expected
- Establishes the visual design, typography, and layout standards
- Contains all key sections that generated dossiers should strive to match:
  - Profile header with contact links and investment metadata
  - Background with career highlights and unique differentiators
  - Investment thesis with key concepts and philosophy
  - Portfolio pattern recognition with company tables
  - Recent activity (podcasts, articles)
  - Positioning framework with templates and analogies
  - Anticipated Q&A with counter-arguments
  - Meeting prep checklist
  - Interactive Company Positioning Analysis feature

**When to Reference**:
- When designing new dossier templates
- When improving the AI generation prompts
- When evaluating the quality of generated dossiers
- When adding new sections or features to dossiers

**Archived**: February 4, 2026
